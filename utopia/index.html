Project Utopia

Project Utopia is a procedural island generator, written around the Unity Burst compiler and Job system.

This project is currently being redeveloped to allow for further customisation of worlds, along with a better noise algorithm.

After this redevelopment, the project will be utilised in an open world racing game, similar to games such as Project Burnout, Forza Horizon and Test Drive Unlimited.
