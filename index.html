Project Utopia

A procedurally generated racing game,
designed around the Unity Job System and Burst Compiler.

This project was designed to be a procedural generator which was quick in execution whilst being easy to use and extend.

Now, the project is being built into a full procedurally generated racing game, along with tweaks to the generator to improve usability/performance.
